# Creator Workflow Template

A repeatable weekly content workflow template for independent creators using creator-tools.

---

## Weekly Content Workflow

### Monday — Plan
- [ ] Review last week's analytics (views, engagement, revenue)
- [ ] Choose 3 content topics for the week
- [ ] Run topics through **Metadata Generator** to validate SEO potential
- [ ] Draft video outlines or blog post structures

### Tuesday — Script & Write
- [ ] Write full scripts for video content
- [ ] Run script through **Script Analyzer** for hook strength + retention risk
- [ ] Apply suggested improvements
- [ ] Write Shopify product descriptions if launching new products

### Wednesday — Create
- [ ] Film / record content
- [ ] Edit videos
- [ ] Prepare thumbnails and cover images

### Thursday — Optimize & Schedule
- [ ] Generate metadata (titles, descriptions, tags) using **Metadata Generator**
- [ ] Schedule posts across platforms (YouTube, TikTok, Instagram)
- [ ] Queue Shopify listings with generated descriptions

### Friday — Publish & Engage
- [ ] Publish scheduled content
- [ ] Respond to comments for the first 2 hours (boosts algorithm)
- [ ] Cross-promote across platforms

### Weekend — Learn & Iterate
- [ ] Review performance data
- [ ] Note what worked and what didn't
- [ ] Update templates and prompt configurations for next week

---

## Tips
- Consistency beats perfection — ship on schedule
- Use creator-tools templates to reduce decision fatigue
- Batch similar tasks together for efficiency
