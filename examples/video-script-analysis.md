# Example: Video Script Analysis

## Input

A 10-minute video script about how beginners can start using AI tools for online selling.

---

## Output (AI Analysis)

### Hook Strength: 6/10
The opening 30 seconds mentions "AI tools" but lacks a specific pain point or curiosity gap. Suggested rewrite:
> "Most sellers waste 3 hours a day on tasks AI can do in 3 minutes — here's the exact workflow."

### Audience Retention Risk Points
- **2:15** — Technical jargon spike ("API integration") — consider simplifying or adding a visual
- **5:40** — Pacing slows, no new information for 45 seconds — add a case study or demo
- **8:10** — Conclusion starts too early, viewer drop-off predicted

### Suggested Chapter Structure
1. Hook + Promise (0:00–0:45)
2. The Problem (0:45–2:00)
3. Tool #1 Walkthrough (2:00–4:30)
4. Tool #2 Walkthrough (4:30–7:00)
5. Real Results / Social Proof (7:00–9:00)
6. CTA + Next Steps (9:00–10:00)

### CTA Improvement
Current: "Subscribe if you liked this"
Suggested: "Download the free AI toolkit in the description — I update it every month"

### Title Ideas
- "I Used AI to Run My Entire Shopify Store for 30 Days (Here's What Happened)"
- "5 AI Tools That Made Me $8K Last Month Selling Online"

### Description
AI tools for online sellers are changing the game in 2026. In this video, I walk through the exact workflow I use to automate product listings, write descriptions, and plan content — all with free tools. Perfect for beginners.
