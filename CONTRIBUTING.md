# Contributing to creator-tools

Thank you for your interest in contributing! creator-tools is an open-source toolkit for independent content creators, and we welcome contributions of all kinds.

## How to Contribute

### Reporting Issues
- Use [GitHub Issues](https://github.com/fei1098046776-tech/creator-tools/issues) to report bugs or request features
- Please include a clear description and steps to reproduce

### Submitting Pull Requests
1. Fork the repository
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Commit with a clear message: `git commit -m "Add: description of change"`
5. Push to your fork: `git push origin feature/your-feature-name`
6. Open a Pull Request against `main`

### What We Welcome
- New creator tool templates
- AI prompt improvements
- Documentation and examples
- Bug fixes
- Translations (especially for non-English creators)

## Code Style
- Keep it simple — many of our users are non-technical creators
- Comment your code clearly
- Prefer vanilla JS over frameworks for static tools

## Community
This project is maintained by independent creators, for independent creators. Be kind, be constructive.
