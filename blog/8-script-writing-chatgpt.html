The Prompt Framework

Use this prompt: 'Act as a YouTube Shorts expert. Write a 150-word script. Start with a contrarian hook. Use short, punchy sentences. End with a subtle call to action.'

Always Humanize the Output

Never copy-paste directly. AI struggles with natural speaking rhythm. Read the script out loud and change complex words into casual, conversational language.
