How to Build a Content Planner in Notion

Published: [DATE]

Consistency is key to growing an audience. A content planner keeps you organized so you never scramble for video ideas. Notion is the ultimate tool for this. Here is how to set up a clean board.

1. Create a Database Table

Start with a fresh page and type `/table-database`. This will hold all your video projects.

2. Define Crucial Properties

Add properties to track the status of each project:

  • Status (Select): Idea, Scripting, Filming, Editing, Ready, Published.
  • Platform (Multi-select): YouTube, TikTok, Blog, Newsletter.
  • Publish Date (Date): Target release date.
  • URL (URL): Link to live post.

3. Set Up a Kanban Board View

Add a new database view and select 'Board.' Group by the 'Status' property. Now you can drag cards from 'Idea' to 'Published' as you work, providing a satisfying workflow.

4. Write Scripts Directly in the Cards

Click open any card to use it as a document editor. Write your hook, outline, and script right inside the card to keep all assets in one place.
